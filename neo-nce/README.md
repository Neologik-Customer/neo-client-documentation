## 🎯 What is Neo NCE?

Neo NCE (Neologik Conversational Engine) is a modern React-based user interface for managing AI agents, knowledge indexes, and data connections. It provides an intuitive interface for:

- **Agent Management**: Create and configure AI agents with customizable instructions, plugins, and connections
- **Knowledge Management**: Upload documents, manage search indexes, and connect to external data sources
- **Data Integration**: Connect to SQL databases, SharePoint, Confluence, and Azure Blob Storage
- **Accessibility**: Comprehensive ARIA attributes and keyboard navigation support

## 🔧 Technical Stack

- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4.0
- **Authentication**: Azure AD (MSAL)
- **UI Components**: Radix UI primitives
- **Icons**: Lucide React
